# Graph Theory and Algorithms

This repository contains implementations, studies, and notes on graph theory concepts and algorithms.

## Topics Covered

- Graph representations (adjacency matrix, adjacency list)
- Graph traversal algorithms (BFS, DFS)
- Shortest path algorithms (Dijkstra's, Bellman-Ford, Floyd-Warshall)
- Minimum spanning trees (Kruskal's, Prim's)
- Network flows
- Graph coloring
- Topological sorting
- Strongly connected components

## Structure

- `/algorithms/` - Implementation of various graph algorithms
- `/notes/` - Theoretical notes and explanations
- `/examples/` - Example problems and solutions
- `/visualizations/` - Graph visualization tools and examples
